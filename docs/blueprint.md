# **App Name**: UtsavConnect

## Core Features:

- Home Screen Hub: Home Screen: Display countdowns, quick-access icons, and a hero image of Lord Ganesha.
- Live Darshan Streaming: Live Darshan: Stream multi-angle video with aarti lyrics, music, and night mode.
- Interactive Event Schedule: Event Schedule: Display event schedule with RSVP and reminder options.
- Real-Time Community Chat: Live Chat: Offer real-time chat for devotees to share memories and wishes; text, emojis, voice notes & photo sharing are enabled
- Photo Gallery: Gallery: Allow users to upload, share and download event photos. Personal upload feature.
- Digital Donation Platform: Donation & Support: Enable UPI/QR code donations and track fund usage for transparency.
- Blessings Wall Translator: Blessings Wall Translation Tool: Using an LLM, detect when the text from one user in the Blessings Wall would likely be beneficial for a second user, and offer a translation of the message in the first user's language into the second user's.

## Style Guidelines:

- Primary color: Saffron (#FF9933) to represent the cultural and spiritual essence of Ganesh Utsav.
- Background color: Light beige (#F5F5DC), providing a neutral, bright, and clean backdrop that doesn't distract from the content.
- Accent color: Deep blue (#336699) to highlight interactive elements and important information, drawing attention effectively.
- Font: 'Inter' (sans-serif) for both headings and body text, due to its clean, modern, neutral appearance.
- Use culturally relevant icons with a touch of modernity to represent different sections and actions.
- Clean, card-based layout for easy navigation and content prioritization.
- Subtle animations on icon taps and screen transitions to enhance user experience.
